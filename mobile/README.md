# NexusRM Mobile

The hackathon MVP focuses on the responsive React web app so the core SaaS workflows stay polished and demo-ready.

Future Expo app scope:

- Expo Router tabs for Dashboard, Clients, Deals, Tasks and AI
- Secure token storage with Expo SecureStore
- Native push notifications for overdue tasks and AI risk alerts
- Offline client notes with SQLite sync queue
- Mobile-first deal update and follow-up generation flows
